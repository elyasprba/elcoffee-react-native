## üìå How To Install?

- Clone This Repo

```
$ git clone https://github.com/elyasprba/elcoffee-react-native.git
```

- Go To Folder Repo

```
$ cd el-coffeeshop
```

- Setup .env

```
REACT_APP_HOST
```

- Install Module

```
$ npm install
```

- To Start The Project

```
$ npm run startDev
```
